Open Graph Default Image
========================

You need to create a default Open Graph image named "og-default.png" in this directory.

Recommended specifications:
- Size: 1200x630 pixels (1.91:1 ratio)
- Format: PNG or JPG
- File size: Under 5MB
- Content: Your name, tagline, or a branded image

This image will be displayed when your website links are shared on:
- Facebook
- Twitter/X
- LinkedIn
- Slack
- Discord
- And other social media platforms

Create a simple image with:
- Your name "Nishant Arora"
- Tagline "Software Engineer / Builder / Learner"
- Clean background matching your website theme

You can use tools like:
- Canva (https://www.canva.com)
- Figma
- Adobe Photoshop
- Or any image editor

Once created, save it as: images/og-default.png
